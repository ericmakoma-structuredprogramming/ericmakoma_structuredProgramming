This repository contains the source code for my C programming assignment.
Each task focuses on basic programming concepts such as arithmetic operations, conditional statements, loops, and functions.
The code is written in C and compiled using GCC.
Task 1:Simple calculator that performs addition, subtraction, multiplication, and division on two numbers.
Task 2:Program to calculate the surface area of a sphere given the radius.
Task 3:Program to ask the user to enter a string, display the string back, and find & display the length of the string.
